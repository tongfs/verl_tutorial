## 一、第一部分：感知机与多层神经网络（约 1 小时）

### 1.1 必学知识点清单

| 知识点 | 说明 | 与 verl / 深度学习的关系 |
|--------|------|---------------------------|
| 神经元 | 接收输入、加权求和、经激活函数输出 | 神经网络的基本单元 |
| 激活函数 | 引入非线性，如 ReLU、Sigmoid | 没有激活函数 = 只能拟合线性 |
| 全连接层 | 每个输入连到每个输出，即矩阵乘法 + 偏置 | 最基础的层结构 |
| 多层神经网络 | 多个全连接层堆叠，输入→隐藏层→输出 | LLM 也是多层堆叠 |
| 前向传播 | 输入从左到右逐层计算，得到最终输出 | 推理和训练都要做 |

### 1.2 建议学习顺序

1. **从感知机到神经元**（15 分钟）  
   - 感知机：输入 x₁, x₂ → 加权 w₁x₁ + w₂x₂ + b → 输出 0 或 1  
   - 神经元：同上，但输出可连续（经激活函数）  
   - 练习：画一个 3 输入 1 输出的神经元，写出输出公式

2. **激活函数**（15 分钟）  
   - Sigmoid：σ(x) = 1/(1+e⁻ˣ)，输出 (0,1)，用于二分类概率  
   - ReLU：max(0, x)，最常用，解决梯度消失  
   - 为什么需要：线性变换的复合仍是线性，无法拟合复杂函数  
   - 练习：画出 ReLU 的曲线，理解「负值变 0」

3. **全连接层（Dense / Linear）**（15 分钟）  
   - 公式：y = Wx + b，W 是权重矩阵，b 是偏置  
   - 输入维度 → 输出维度：如 (batch, 784) @ (784, 256) → (batch, 256)  
   - 练习：若输入 10 维、输出 5 维，W 和 b 的形状各是多少？

4. **多层神经网络**（10 分钟）  
   - 输入层 → 隐藏层 1 → 隐藏层 2 → … → 输出层  
   - 每层：线性变换 + 激活函数（最后一层有时不加）  
   - 前向传播：x → h₁=σ(W₁x+b₁) → h₂=σ(W₂h₁+b₂) → … → 输出  
   - 练习：写出 2 层网络（输入→隐藏→输出）的前向公式

5. **与 Day 2 的衔接**（5 分钟）  
   - 每层的 W、b 都是可学习参数  
   - 训练目标：调整 W、b 使 loss 最小  
   - 需要梯度：对每个参数求 ∂loss/∂W、∂loss/∂b → 用到反向传播

## 二、第二部分：反向传播与梯度下降（约 1 小时）

### 2.1 必学知识点清单

| 知识点 | 说明 | 与 verl / 深度学习的关系 |
|--------|------|---------------------------|
| Loss 损失函数 | 衡量预测与真实的差距，如 MSE、交叉熵 | 训练的目标是最小化 loss |
| 反向传播 | 从输出层往输入层，用链式法则算每层参数的梯度 | PyTorch 的 backward() 自动完成 |
| 梯度下降 | 用梯度更新参数：θ ← θ − lr × ∇θ | 优化器（如 SGD、Adam）在做的事 |
| 优化器 | 决定如何用梯度更新参数 | verl 训练中会配置 optimizer |

### 2.2 建议学习顺序

1. **Loss 损失函数**（10 分钟）  
   - 回归：MSE = (1/n)Σ(y_pred − y_true)²  
   - 分类：交叉熵 CrossEntropy  
   - 目标：loss 越小，模型越好  
   - 练习：若 y_true=1，y_pred=0.8，MSE 是多少？（0.04）

2. **为什么需要反向传播**（15 分钟）  
   - 问题：网络有很多层、很多参数，如何高效算每个参数的梯度？  
   - 暴力：对每个参数单独求导 → 计算量爆炸  
   - 反向传播：从输出往输入，利用链式法则，每层只算一次，复用中间结果  
   - 直观：误差从输出层「反向」传播到每一层，每层知道「我该往哪改」  
   - 练习：能用自己的话解释「为什么需要反向传播」

3. **反向传播的直观理解**（15 分钟）  
   - 输出层：先算 ∂loss/∂输出  
   - 倒数第二层：∂loss/∂该层输出 = 后一层传过来的梯度 × 该层对输入的导数  
   - 逐层往前，每层得到自己参数的梯度  
   - 3Blue1Brown 的动画：看「误差如何一层层传回去」  
   - 练习：能画出「梯度从输出到输入」的传播方向

4. **梯度下降与参数更新**（10 分钟）  
   - 有了梯度 ∇θ，更新：θ ← θ − 学习率 × ∇θ  
   - 学习率：步长，太大发散、太小收敛慢  
   - 重复：前向算 loss → 反向算梯度 → 更新参数，直到收敛  
   - 练习：写出梯度下降的一步更新公式

5. **训练循环全景**（10 分钟）  
   - for epoch in ...:  
   - &nbsp;&nbsp;&nbsp;&nbsp;for batch in ...:  
   - &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;前向：output = model(input)  
   - &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;算 loss：loss = criterion(output, target)  
   - &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;反向：loss.backward()  
   - &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;更新：optimizer.step()  
   - 练习：能按顺序说出训练循环的 4 个步骤

## 三、自测题

### 3.1 神经网络自测

1. 一个全连接层：输入 4 维，输出 3 维。写出 W 和 b 的形状。
2. 为什么需要激活函数？如果不用激活函数，多层网络等价于什么？
3. 写出 2 层网络（输入 x → 隐藏层 h → 输出 y）的前向公式，隐藏层用 ReLU。

### 3.2 反向传播自测（重点）

1. **为什么需要反向传播？** 用自己的话写 2–3 句解释。
2. **训练循环四步**：请按正确顺序写出：前向、算 loss、反向、更新。
3. **梯度下降**：若当前参数 θ=5，梯度 ∇θ=2，学习率 lr=0.1，更新后的 θ 是多少？  
   **答案**：θ_new = 5 − 0.1×2 = 4.8

### 3.3 参考答案

2. 前向 → 算 loss → 反向（backward）→ 更新（optimizer.step）  
3. θ_new = 4.8
