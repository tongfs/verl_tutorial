## 一、第一部分：线性代数基础（约 1 小时）

### 1.1 必学知识点清单

| 知识点 | 说明 | 与 verl / 深度学习的关系 |
|--------|------|---------------------------|
| 向量 | 有方向、有大小的量，可写成列或行 | 特征、embedding、logits 都是向量 |
| 矩阵 | 数字的二维排列，可看作「一组列向量」 | 权重矩阵 W、注意力矩阵 |
| 矩阵乘法 | 左行乘右列，得到新矩阵 | 神经网络里全是矩阵乘法 |
| 矩阵与线性变换 | 矩阵左乘向量 = 对空间做变换 | 理解「层」在做什么 |
| 转置 | 行变列、列变行 | 维度对齐、梯度形状 |

### 1.2 建议学习顺序

1. **向量是什么**（12 分钟）  
   - 几何：箭头；代数：有序数字列表  
   - 向量加法、数乘  
   - 练习：在纸上画两个向量，做加法

2. **线性组合与张成的空间**（12 分钟）  
   - 线性组合：`a·v₁ + b·v₂`  
   - 张成的空间：所有线性组合构成的集合  
   - 练习：两个不共线的二维向量张成整个平面

3. **矩阵与线性变换**（15 分钟）  
   - 线性变换：保持网格线平行且等距分布的变换  
   - 矩阵：描述线性变换的「数字表格」  
   - 矩阵左乘向量 = 得到变换后的向量  
   - 练习：理解 2×2 矩阵如何把 (1,0) 和 (0,1) 变到新位置

4. **矩阵乘法**（15 分钟）  
   - 两个矩阵相乘 = 两个线性变换的「复合」  
   - 计算规则：结果第 i 行第 j 列 = 左矩阵第 i 行 · 右矩阵第 j 列  
   - 形状：(m×n) @ (n×p) = (m×p)  
   - 练习：手算一个 2×2 乘 2×2

5. **转置**（6 分钟）  
   - 行变列：`A[i,j]` → `A.T[j,i]`  
   - 用途：维度对齐，如 (batch, dim) 与 (dim, hidden) 相乘  
   - 练习：写出 2×3 矩阵的转置形状

## 二、第二部分：微积分与梯度（约 1 小时）

### 2.1 必学知识点清单

| 知识点 | 说明 | 与 verl / 深度学习的关系 |
|--------|------|---------------------------|
| 导数 | 函数在某点的变化率、切线斜率 | loss 对参数的导数 = 更新方向 |
| 偏导数 | 多元函数对某一个变量的导数 | 对每个参数分别求导 |
| 梯度 | 所有偏导数组成的向量，指向函数增加最快的方向 | 梯度下降 = 沿梯度反方向更新 |
| 链式法则 | 复合函数求导：外层导数 × 内层导数 | 反向传播的数学基础 |

### 2.2 建议学习顺序

1. **导数回顾**（10 分钟）  
   - 导数 = 瞬时变化率 = 切线斜率  
   - 常见导数：`x²` → `2x`，`x` → `1`，常数 → `0`  
   - 练习：写出 `f(x)=x²` 在 `x=3` 处的导数

2. **多元函数与偏导数**（15 分钟）  
   - 多元函数：如 `f(x, y) = x² + y²`  
   - 偏导数：对 x 求导时把 y 当常数，对 y 求导时把 x 当常数  
   - ∂f/∂x = 2x，∂f/∂y = 2y  
   - 练习：求 `g(x,y)=xy` 的 ∂g/∂x 和 ∂g/∂y

3. **梯度**（15 分钟）  
   - 梯度 = (∂f/∂x, ∂f/∂y, …)，是一个向量  
   - 几何意义：指向函数值增加最快的方向  
   - 梯度下降：参数 ← 参数 − 学习率 × 梯度  
   - 练习：写出 `f(x,y)=x²+y²` 在点 (1,2) 处的梯度

4. **链式法则**（15 分钟）  
   - 若 `z = f(y)` 且 `y = g(x)`，则 dz/dx = (dz/dy)·(dy/dx)  
   - 多元情形：复合时每个分支的偏导相乘再相加  
   - 反向传播 = 从输出层往输入层，一层层用链式法则算梯度  
   - 练习：设 `z = y²`，`y = 2x+1`，求 dz/dx（答案：2y·2 = 4(2x+1)）

5. **梯度下降直观**（5 分钟）  
   - 目标：最小化 loss  
   - 每一步：沿梯度反方向走一小步（学习率）  
   - 练习：能用自己的话解释「为什么要沿梯度反方向」

### 2.4 自测与检查

**自测题**  
1. 手算梯度：设 `f(x, y) = x² + y²`，求梯度 ∇f = (∂f/∂x, ∂f/∂y)；在点 (1, 2) 处的梯度是多少？  
2. 链式法则：设 `L = (y_true - y_pred)²`，`y_pred = w·x + b`。若把 L 看作关于 w 的函数，写出 ∂L/∂w 的链式形式（不要求最终展开）。提示：∂L/∂w = (∂L/∂y_pred)·(∂y_pred/∂w)。

<details>
<summary>点击展开参考答案</summary>

- f(x,y)=x²+y² → ∂f/∂x=2x，∂f/∂y=2y → ∇f=(2x, 2y)；(1,2) 处 → ∇f=(2, 4)  
- ∂L/∂w = ∂L/∂y_pred · ∂y_pred/∂w = 2(y_pred−y_true)·x
</details>
